Perlin Noise

Perlin noise is a type of noise function which appears more organic than random. It was developed by Ken Perlin while working on the movie "Tron". In this project, it is used to generate a fluid like motion on some coloured moving particles :)

Difference between Noise and Random function

Perlin noise is smooth and it shows different levels of smoothness at different scales. This nature of perlin noise is used to produce smooth fluid like motion.
